Projects

Federated Learning for Diabetes Prediction

This thesis compares centralized and federated machine learning approaches for diabetes prediction.

The project evaluates predictive performance, probability calibration, explainability and communication efficiency. SHAP is used for interpretation and isotonic regression is used for probability calibration.

Tools: Python, Scikit-learn, SHAP, Federated Learning, Isotonic Calibration

FlyRank Content Refresh Prioritization

I worked with a dataset containing 30,000 content pages and framed content refresh prioritization as a machine learning classification problem.

I created a proxy target, detected target leakage and built a baseline action score using staleness, click-through rate, average position and impressions.

Tools: Python, Pandas, Jupyter Notebook, GitHub

Bitcoin Price Prediction

This project compared multiple machine learning models for Bitcoin price prediction using time-series data.

Models were evaluated using RMSE and MAE, with attention to model complexity, interpretability and practical usefulness.

Tools: Python, Machine Learning, Time-Series Analysis, Data Visualization
